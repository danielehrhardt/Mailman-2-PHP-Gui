Mailman 2 PHP Gui
=======

Beschreibung
------------
Bootstrap PHP Skript zum ein und austragen von E-Mail Adressen aus einer Mailman Mailing Liste.
    
Installation
------------
Host anpassen

    $host = 'yourtld.tdl'; 

Listen eintragen

    $lists = [
       "listone" => "Liste 1",
       "listtwo" => "Liste 2",
    ];
